# Simple Calculator

A clean and efficient application designed to perform fundamental arithmetic operations. This project serves as a practical demonstration of basic programming logic and user input handling.

---

## 🚀 Features

* **Core Operations:** Addition, Subtraction, Multiplication, and Division.
* **Error Handling:** Prevents crashes on invalid inputs or division by zero.
* **Looping:** Allows users to perform multiple calculations without restarting.

## 🛠️ Built With

* **Language:** JavaScript
* **Concepts:** Conditional Logic, Functions, and Error Handling.
